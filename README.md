# 🚀 Portfolio Website (FastAPI)

A personal portfolio website built using FastAPI to showcase my experience, skills, and projects as a Backend Engineer.

## 🔥 Features

- 🧑‍💻 Clean and responsive portfolio UI
- ⚡ FastAPI backend with modular structure
- 📄 Resume download functionality
- 🧩 Jinja2 templating for dynamic rendering
- 🎯 Structured using MVC-style architecture (models, controllers, templates)
- 🛠️ Production-ready deployment (Heroku compatible)

---

## 🛠️ Tech Stack

- **Backend:** FastAPI
- **Templating:** Jinja2
- **Frontend:** HTML, CSS
- **Server:** Uvicorn / Gunicorn
- **Deployment:** Heroku
